# Guruji Astrologer

Astrology app providing horoscope insights, zodiac compatibility, and spiritual guidance.

## ✨ Features
- Daily horoscope predictions
- Zodiac sign details
- Compatibility checker
- Astrology-themed UI

## 🛠 Tech Stack
- Python
- Streamlit
- Google API Client
- Requests, yt-dlp

## 🚀 Deployment
- Hosted on Streamlit Cloud
- Requires `requirements.txt` for dependencies
- Secrets managed via Streamlit Cloud Secrets Manager
